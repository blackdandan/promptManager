# PromptManager Web 前端实现计划

## 项目概述

基于现有项目文档和 API 文档，制定前端实现计划。

## 技术栈确认

- **框架**: Vue 3 + TypeScript
- **状态管理**: Pinia
- **UI组件**: Element Plus
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **构建工具**: Vite

## 核心功能模块

### 1. 用户认证模块
- 用户注册/登录
- 三方登录（GitHub、Google、微信、Apple）
- Token 管理和刷新

### 2. Prompt 管理模块
- Prompt 的增删改查
- 标签管理
- 搜索和筛选
- 收藏功能
- 分类管理

### 3. 数据同步模块
- 多端数据同步
- 离线支持
- 冲突解决

### 4. 分享功能模块
- Prompt 分享
- 公开 Prompt 浏览
- 复制功能

## API 集成计划

### 认证相关 API
- `POST /auth/register` - 用户注册
- `POST /auth/login` - 用户登录
- `POST /auth/refresh` - Token 刷新
- `POST /oauth/callback/{provider}` - 三方登录回调

### Prompt 相关 API
- `GET /prompts` - 获取 Prompt 列表
- `GET /prompts/{id}` - 获取单个 Prompt
- `POST /prompts` - 创建 Prompt
- `PUT /prompts/{id}` - 更新 Prompt
- `DELETE /prompts/{id}` - 删除 Prompt
- `POST /prompts/{id}/favorite` - 切换收藏状态

### 会员相关 API
- `GET /membership` - 获取会员信息
- `POST /membership/upgrade` - 升级会员

## 设计规范分析

基于获取的 Figma 设计数据，以下是详细的设计规范：

### 页面结构
1. **登录页面** - 支持 Google、微信登录和游客模式
2. **首页** - 侧边栏导航 + Prompt 卡片网格布局
3. **搜索页面** - 空状态和搜索结果状态
4. **新建 Prompt 页面** - 包含标题、内容、分类、文件夹、标签等表单
5. **Prompt 详情页** - 展示 Prompt 内容、变量填充、使用统计
6. **设置页面** - 用户信息、账号设置、帮助与支持

### 设计系统
- **主色调**: 蓝色渐变 (#2B7FFF → #9810FA)
- **背景色**: 浅灰色 (#F9FAFB)
- **卡片背景**: 白色 (#FFFFFF)
- **文字颜色**: 深色 (#0A0A0A)、灰色 (#6A7282)
- **边框**: 浅灰色 (#E5E7EB)
- **圆角**: 8px、10px
- **阴影**: 多层阴影效果

### 组件规范
- **按钮**: 主要按钮使用深色背景，次要按钮使用边框
- **输入框**: 浅灰色背景，圆角边框
- **卡片**: 白色背景，边框，阴影，圆角
- **标签**: 小圆角，不同颜色状态
- **侧边栏**: 固定宽度，分隔线
