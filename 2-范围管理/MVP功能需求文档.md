# MVP功能需求文档

## 1. 文档概述

### 1.1 项目背景
基于竞品分析发现，市场上缺乏简洁易用的多平台prompt管理工具。MVP版本将专注于核心功能，提供Web端和Android端的同步体验。

### 1.2 MVP目标
- 3周内上线可用的最小可行产品
- 验证核心功能的市场需求
- 建立基础用户群体
- 收集用户反馈指导后续迭代

### 1.3 目标用户
- **AI重度用户**：需要快速管理和使用prompt
- **内容创作者**：需要创作和分享prompt模板

## 2. MVP功能范围

### 2.1 核心功能模块

```
.
├── Web端功能
│   ├── Prompt管理
│   ├── 模板变量化
│   ├── 搜索和排序
│   └── 数据同步
├── Android端功能
│   ├── Prompt管理
│   ├── 快速复制
│   ├── 离线使用
│   └── 数据同步
└── 服务端功能
    ├── 用户认证
    ├── 数据同步
    └── 基础API
```

## 3. Web端功能需求

### 3.1 用户认证模块
- **用户注册**：邮箱/密码注册
- **用户登录**：邮箱/密码登录
- **记住登录状态**：7天免登录
- **密码重置**：通过邮箱重置密码

### 3.2 Prompt管理模块
- **创建Prompt**：
  - 输入prompt标题和内容
  - 设置标签（最多5个）
  - 选择分类（预设分类：通用、写作、编程、分析、创意）
  - 选择文件夹（可选，支持多级文件夹）
- **编辑Prompt**：修改标题、内容、标签、分类、文件夹
- **删除Prompt**：软删除，可恢复
- **查看Prompt列表**：分页显示，支持卡片视图和列表视图
- **文件夹管理**：
  - 创建、编辑、删除文件夹
  - 支持多级文件夹结构
  - 文件夹拖拽排序
  - 批量移动prompt到文件夹

### 3.3 模板变量化模块
- **变量定义**：支持 {变量名} 格式的变量占位符
- **变量替换**：提供变量输入界面，实时预览替换结果
- **常用变量**：预设常用变量（topic、tone、length、format）

### 3.4 搜索和排序模块
- **关键词搜索**：在标题、内容、标签中搜索
- **标签筛选**：按标签筛选prompt
- **分类筛选**：按分类筛选prompt
- **排序功能**：按创建时间、修改时间、使用频率排序

### 3.5 数据操作模块
- **一键复制**：复制prompt内容到剪贴板
- **分享功能**：生成分享链接（只读视图）
- **使用统计**：记录prompt使用次数

## 4. Android端功能需求

### 4.1 用户认证模块
- **用户登录**：与Web端相同的认证体系
- **自动登录**：支持生物识别登录（指纹/面部）
- **离线模式**：未登录时可本地使用

### 4.2 Prompt管理模块
- **创建Prompt**：简化表单，支持语音输入
- **快速编辑**：支持快捷编辑常用字段
- **批量操作**：支持批量删除、批量添加标签、批量移动文件夹
- **本地存储**：支持离线创建和编辑
- **文件夹管理**：
  - 创建、编辑、删除文件夹
  - 支持多级文件夹结构
  - 文件夹拖拽排序
  - 文件夹内prompt管理

### 4.3 快速使用模块
- **一键复制**：点击即可复制到剪贴板
- **分享到其他应用**：支持分享到微信、QQ、邮件等
- **最近使用**：显示最近使用的prompt
- **收藏功能**：标记常用prompt

### 4.4 搜索模块
- **语音搜索**：支持语音输入搜索关键词
- **快捷搜索**：搜索历史记录
- **标签云**：可视化标签选择

### 4.5 数据同步模块
- **自动同步**：登录后自动同步数据
- **手动同步**：支持手动触发同步
- **冲突解决**：最后修改优先
- **同步状态**：显示同步进度和状态

## 5. 服务端功能需求

### 5.1 用户服务
- **用户注册**：验证邮箱唯一性
- **用户登录**：JWT token认证
- **用户信息**：存储用户基本信息
- **会话管理**：管理用户登录状态

### 5.2 数据服务
- **Prompt存储**：存储用户的所有prompt数据
- **标签管理**：存储标签信息
- **分类管理**：存储分类信息
- **文件夹管理**：存储文件夹结构和权限
- **使用统计**：记录prompt使用数据

### 5.3 同步服务
- **数据同步**：处理多端数据同步
- **冲突检测**：检测数据修改冲突
- **版本控制**：维护数据版本历史
- **增量同步**：只同步变更数据

### 5.4 分享服务
- **分享链接生成**：为prompt生成唯一分享链接
- **分享访问控制**：控制分享内容的访问权限
- **分享统计**：记录分享链接的访问数据

## 7. MVP功能优先级

### 7.1 必须功能（P0）
- Web端：用户登录、Prompt增删改查、一键复制
- Android端：用户登录、Prompt增删改查、一键复制
- 服务端：用户认证、数据存储、基础同步

### 7.2 重要功能（P1）
- Web端：模板变量化、搜索功能、标签管理
- Android端：离线使用、快速搜索、分享功能
- 服务端：数据同步、分享服务

### 7.3 优化功能（P2）
- Web端：高级搜索、批量操作、使用统计
- Android端：语音输入、生物识别、智能推荐
- 服务端：性能优化、监控告警

## 8. 非功能需求

### 8.1 性能需求
- **响应时间**：核心操作<2秒
- **并发支持**：支持1000并发用户
- **数据同步**：同步延迟<30秒

### 8.2 可用性需求
- **Web端**：支持Chrome、Safari、Edge最新版本
- **Android端**：支持Android 8.0及以上版本
- **服务端**：99.5%可用性

### 8.3 数据安全
- **数据传输**：HTTPS加密
- **数据存储**：密码加密存储
- **访问控制**：基于用户的权限控制

## 9. 验收标准

### 9.1 功能验收
- [ ] 用户可以在Web端和Android端注册登录
- [ ] 用户可以在两端创建、编辑、删除prompt
- [ ] 数据可以在两端实时同步
- [ ] 支持模板变量替换功能
- [ ] 支持一键复制和分享功能

### 9.2 用户体验验收
- [ ] 核心功能操作流程顺畅
- [ ] 界面简洁易用，学习成本低
- [ ] 数据同步稳定可靠
- [ ] 错误提示清晰明确

### 9.3 性能验收
- [ ] 页面加载时间<3秒
- [ ] 数据同步延迟<30秒
- [ ] 支持离线使用基本功能
- [ ] 应用崩溃率<1%
